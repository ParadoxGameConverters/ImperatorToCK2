This program will convert an Imperator: Rome save game to a Crusader Kings 2 mod.

----
IMPORTANT LINKS
----

Forum thread: http://forum.paradoxplaza.com/forum/ --- 
Github project: https://github.com/ParadoxGameConverters/ImperatorToCk2
Windows development build: https://github.com/ParadoxGameConverters/ImperatorToCk2/releases/tag/WindowsRelease

----
REQUIREMENTS
----
Imperator - N/A
CK2 - N/A

-----
INSTRUCTIONS
-----
1. Run Frontend.exe
2. Set all the options, making sure the directories are correct.
3. Click Convert
4. Start CK2 and select the mod.


----
MANUAL METHOD INSTRUCTIONS
----

1. Download the converter and unzip it wherever.
2. Edit configuration.txt to point at your game installs, your savegame location, and to set any options you care about.
3. Run ImperatorToCk2.exe
4. Go into the output folder, then the subfolder named after your save.
5. Copy the folder named after your save and the corresponding mod file into your CK2 mod folder.
6. Start CK2, and select the mod.
